## User Manual

### How to fix pom.xml: Dependency 'org.aspectj:aspectjweaver:1.9.25.1' not found

 ```mvn dependency:purge-local-repository```\
 ```mvn clean install -U```
 
### How to enable libraries sources and documentation

1. Open Maven tab
2. Right-click project name
3. Click 'Download Sources and Documentation'
